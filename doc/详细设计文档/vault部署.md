## 关键信息

在服务第一次启动的时候，在控制台，会显示Unseal Key和Initial Root Token，需要立即保存。
| 关键项 | 详细描述 | 备注 |
| ----  | ----  | ----  |
| Unseal Key | 多个，出于安全考虑，需要分给不同的人保管 ||
| Root Token | 权限相当于Linux的root用户权限，需要妥善保存 ||
